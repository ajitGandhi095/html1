Git & Github :-
=========================
    --> Git is a application which is used for version control. git is a centralized version control system. git keeps track of all your versions, changes and update.

    --> git will contain 3 phases :-
                        1) Working Phase/Working Area :- git init
                        2) Staging Phase/Staging Area :- git add .(Everything add) / 
                                                    git add folder_name (one folder add)
                        3) Local Repository :- git commit -m "message"
                        (if any error coming  who are u)
        -->To solve who r u -> git config --global user.email "your email id"
                            -> git config --global user.name "git username"

        --> Github is a platform created by microsoft where developers share, depoly and 
            merge code.

    How to push your code Github
    ------------------------------
    --> git remote add origin "code_link paste here" -> it is run only one time in one git folder.
    --> git push -u origin main/master(as your git brance name here).
    --> Again add, commit.
